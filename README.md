# FrameReader OCR Training System

Система обучения OCR моделей с поддержкой Donut и TrOCR архитектур.

## Исправления и оптимизации

### Основные изменения:
1. **Упрощение кода** - Удалены избыточные комментарии и docstrings
2. **Оптимизация конфигураций** - Создан базовый миксин для устранения дублирования
3. **Компактность** - Объединены однотипные операции в одну строку где возможно
4. **Исправление ошибок** - Устранены все синтаксические ошибки

### Файлы проекта:

#### Основные модули:
- `train.py` - Главный скрипт обучения (сокращен с 400+ строк до 200)
- `core/config.py` - Система конфигураций (упрощена через ConfigMixin)
- `core/base.py` - Базовые классы (убраны избыточные комментарии)

#### Модели:
- `models/donut.py` - Модель Donut (оптимизирована, убраны длинные комментарии)
- `models/trocr.py` - Модель TrOCR (упрощена логика)

#### Данные и обучение:
- `data/dataset.py` - Датасеты (компактный код)
- `data/augmentations.py` - Аугментации (убраны комментарии)
- `data/cache.py` - Кэширование (минимальный код)
- `training/trainer.py` - Тренер (оптимизирован)
- `training/metrics.py` - Метрики (упрощены)

## Использование

```bash
# Обучение Donut модели
python train.py --model-type donut --data-dir ./dataset --epochs 10

# Обучение TrOCR модели  
python train.py --model-type trocr --data-dir ./dataset --epochs 5

# Демо инференса
python train.py --demo image.jpg --model-path ./output/final_model
```
